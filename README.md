### Leitor de Planilhas (Excel)
* **O que ele faz?** <br>O script usa a poderosa biblioteca `pandas` para ler um arquivo Excel (`.xlsx`) e carregar seus dados para dentro do Python, exibindo-os no terminal.

* **Como Usar:**
  <br>1- Instale as bibliotecas necessárias: pip install pandas openpyxl<br>
  2- Crie um arquivo dados.xlsx na mesma pasta.<br>
  3- Execute o script (python leitor_excel.py) para ler o arquivo e exibir seu conteúdo.
